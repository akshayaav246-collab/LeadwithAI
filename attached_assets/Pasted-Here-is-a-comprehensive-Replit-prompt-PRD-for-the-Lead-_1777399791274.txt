# Lead with AI — Full Website PRD for Replit

## Project Overview

Build a 3-page marketing website for **Lead with AI**, a 2-day professional AI program by **Global Knowledge Technologies (GKT)**. The site must be production-quality, visually refined, and fully navigable with smooth transitions.

---

## Tech Stack

- **Framework**: React (Vite)
- **Routing**: React Router v6
- **Styling**: Plain CSS (no Tailwind, no CSS modules — one global `styles.css`)
- **Fonts**: Import from Google Fonts — `Playfair Display` (serif, for headings) and `EB Garamond` (serif, for body text). Load both via `<link>` in `index.html`.
- **Animations**: CSS transitions and keyframes only. No animation libraries.
- **Icons**: Inline SVG only. No icon libraries.
- **No external UI component libraries.**

---

## Navigation (Persistent across all pages)

The `<nav>` component is sticky (top: 0, z-index: 1000). It has a white background with a 1px bottom border in `--color-sand`. On scroll past 60px, add class `nav-scrolled` which adds a subtle box-shadow (`0 2px 16px rgba(59,47,47,0.08)`).

**Left side**: Logo text — "Lead with AI" in Playfair Display, 1.2rem, bold, `--color-espresso`. Beneath it in smaller text: "by Global Knowledge Technologies" in `--color-stone`, 0.7rem.

**Right side nav links**: `Home`, `Program`, `Speakers`, `Register Now` — the last one styled as a small `btn-primary` pill.

**Mobile**: Hamburger menu below 768px. Clicking hamburger opens a full-width dropdown panel that slides down with a CSS max-height transition (0 → auto via max-height: 400px, transition 0.35s ease). Menu links stack vertically with 1.5rem padding each.

**Active link**: Underlined with a 2px `--color-sienna` line using `border-bottom`.

**Page transitions**: Wrap all `<Routes>` in a container with class `page`. On route change, apply `fade-in` keyframe:
```css
@keyframes fade-in {
  from { opacity: 0; transform: translateY(12px); }
  to   { opacity: 1; transform: translateY(0); }
}
.page { animation: fade-in 0.4s ease forwards; }
```
