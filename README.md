AI project

## Overview

This repository contains the "Lead with AI" project, separated into `frontend` and `backend` directories.

## Project Structure

- **`frontend/`**: A React + Vite application that serves as the marketing website for the "Lead with AI" 2-day professional AI program. 
- **`backend/`**: A placeholder directory reserved for the future Node.js + Nest.js backend with MongoDB.

## Running Locally

### Frontend
To run the frontend marketing website locally:

1. Navigate to the frontend directory:
   ```bash           
   cd frontend
   ```

2. Install dependencies (if you haven't already):
   ```bash
   npm install
   ```

3. Start the Vite development server:
   ```bash
   npm run dev
   ```

The frontend application uses modern React and plain CSS for its design system.
